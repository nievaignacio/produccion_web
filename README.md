# Producción Web | Escuela Da Vinci (2022)

Este repositorio contiene los recursos y el proyecto central de la materia **Producción Web**, dictada en la carrera de Analista de Sistemas. El enfoque es puramente práctico, centrado en la implementación del patrón arquitectónico **MVC** en **PHP**.

El repositorio se divide en dos secciones principales:

## 📁 POO (Programación Orientada a Objetos)
Contiene ejemplos prácticos y ejercicios sobre los fundamentos del lenguaje PHP bajo el paradigma de objetos.

## 📁 MVC (Custom Framework)
Esta carpeta contiene la estructura del proyecto abordado. El objetivo principal del mismo es demostrar la implementación del patrón **Modelo-Vista-Controlador (MVC)** sin dependencias externas. Creando un "Framework  personalizado" que demuestre el funcionamiento interno de una arquitectura web profesional. 

### Características Principales
- **Arquitectura desacoplada:** Separación total de lógica de negocio, ruteo y presentación.
- **Programación Orientada a Objetos:** Uso intensivo de clases, herencia y abstracción.
- **Seguridad:** Implementación de PDO para la prevención de SQL Injection.
- **Clean Code:** Estructura de carpetas profesional lista para escalar.
